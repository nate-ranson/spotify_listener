Destroy The Machines

This is a mini project that uses Python, MySQL, and a little bit of jQuery to render tables for updated stats. Every minute, a cron job kicks off that runs a Python script to query the Spotify API. If it finds I'm currently listening to something, it'll log the artist, track, and album and write it to a MySQL table.

The front end is written with Python's Flask. There are a handful API endpoints written to feed the jQuery a JSON object with the details of the tables. You can hit those directly at:

  • https://destroythemachin.es/api/artists
  • https://destroythemachin.es/api/tracks
  • https://destroythemachin.es/api/albums
  • https://destroythemachin.es/api/latest
